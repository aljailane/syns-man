طرق التحديث

1) التحديث التلقائي داخل التطبيق

يعتمد على electron-updater ويتطلب Release صالح في GitHub.

  1. فحص التحديث.
  2. تنزيل الإصدار الجديد.
  3. إظهار التقدم في واجهة About.
  4. التثبيت عبر زر Install Update.

2) التحديث اليدوي

تحميل نسخة Setup أو Portable من صفحة Releases ثم التثبيت يدويًا.

3) الملفات المطلوبة لنجاح التحديث التلقائي

  • Setup EXE
  • blockmap
  • latest.yml

4) ملاحظات مهمة

التحديث التلقائي لا يعمل في npm start (وضع التطوير).

5) استكشاف مشاكل شائعة

تحقق من رقم الإصدار، ملفات Release، وصحة إعدادات build.publish.
