دليل رفع المشروع إلى GitHub

رفع الكود

  1. git add .
  2. git commit -m "prepare release"
  3. git push

بناء EXE

نفّذ: npm run dist

رفع Release يدوي

  1. أنشئ Tag مثل v1.0.2.
  2. ارفع ملفات dist المهمة (Setup + blockmap + latest.yml + portable).

رفع Release آلي

استخدم npm run release:publish بعد ضبط GH_TOKEN.
